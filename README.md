# クロードくん (Claude Crab)

Claudeの公式マスコット「Claude the Crab」をモチーフにした、ピクセルアート風のデスクトップペット風Webアプリ。
カニがブラウザ内をうろうろ・寝る・ジャンプ・振り返るなどしながら、最新のAIニュースをティッカーで表示し、チャットで会話もできます。

![status](https://img.shields.io/badge/status-WIP-orange)

---

## デモ / 使い方

1. `claude_crab.html` をブラウザで開くだけで動作します（追加インストール不要）。
2. 画面いっぱいにカニの世界が表示されます。
3. 上部の `GROQ API KEY` 欄に [Groq](https://console.groq.com/) のAPIキー（無料）を入力して `SAVE` すると、チャット機能が有効になります。
   - キーは `localStorage` にブラウザ内保存されるのみで、外部送信はされません。
   - APIキーがなくても固定セリフでの動作・RSSニュース取得は可能です。

## 主な機能

### 🦀 カニの生活シミュレーション
- **WALK**：通常速度で歩く
- **WALK_SLOW**：眠そうにゆっくり歩く
- **STOP**：立ち止まって固定セリフを吹き出しで表示
- **TURN**：びっくりして振り返る（方向転換）
- **JUMP**：ジャンプ（重力アニメーション）
- **SLEEP**：寝る（ZZZ表示）

各アクションに応じて表情（通常／驚き／眠い／寝／怒り／嬉しい／やられ）が変化します。

### 🕐 時報・時間帯シーン（24時間対応）
- 毎時0分にカニが立ち止まり、その時間帯に応じたセリフ・表情・時報音を再生
- 起床・歯磨き・朝食・通勤・仕事・昼食・コーヒーブレイク・退勤・帰宅・お風呂・夕食・おやすみ…など24パターン
- 背景の空の色・太陽/月の位置・星・雲・建物の窓の灯りが時間帯に応じて自動変化
- 深夜帯はランダムで流れ星が発生

### 🌤️ 常時ステータス表示
- 現在時刻（大きく表示）・日付・曜日
- 東京の天気・気温（[Open-Meteo](https://open-meteo.com/) APIより取得、APIキー不要）
- 現在の「行動」ラベル（Working / Lunch / Bath など）

### 📡 ニュースティッカー
- Anthropic公式・TechCrunch AI・The Verge AI・ITmedia AI+ など複数のRSSフィードから最新ニュースを取得
- クリックで元記事を新規タブで開く
- 🔄 RELOADボタンで再取得
- RSS取得できない場合は固定ニュースセットにフォールバック

### 💬 チャット機能
- 下部の入力欄からカニに話しかけられる
- Groq API（`llama-3.1-8b-instant`）でリアルタイム応答
- 日本語IME変換中のEnterキーによる誤送信を防止
- 返答の吹き出しはキー入力・クリックがあるまで消えない
- 📋 LOGボタンで会話履歴を確認・クリア可能

### 🎵 効果音
- Web Audio API（矩形波・三角波）によるファミコン風効果音
- ♪ボタンでミュート切り替え

### 🌐 日英切り替え
- 🌐 JP / EN ボタンでつぶやき・ニュース・チャットの言語を切り替え
- 時計・日付・天気・行動ラベルは常に英語表記（Press Start 2P フォント統一）

## 技術構成

- **フロントエンド**: 単一HTMLファイル（HTML + CSS + Vanilla JS）
- **フォント**: Google Fonts（Press Start 2P / M PLUS Rounded 1c）
- **背景**: SVGによるドット絵の街並み（16:9基調、`preserveAspectRatio="xMidYMid slice"`で画面全体に対応）
- **チャットAPI**: [Groq API](https://groq.com/)（`llama-3.1-8b-instant`、無料枠あり）
- **天気API**: [Open-Meteo](https://open-meteo.com/)（APIキー不要）
- **ニュースRSS**: rss2json.com経由でCORS回避

## 既知の制約 / 今後の展望

- Anthropic（Claude）のAPIはブラウザからCORS制限により直接呼び出せないため、チャットはGroq APIを使用
- 定時（8時・12時・17時など）の自動ニュース更新は、Webアプリとして常時起動している場合のみ有効（現状はRELOADボタンによる手動更新が基本）
- 時間帯シーンは現在「背景色・表情・セリフ・アイテム絵文字」での演出。専用イラストでの背景切り替えは今後の拡張候補
- Vercel等へのデプロイにより、定時実行（cron）やAPIキーのサーバーサイド管理が可能になる
